# esx_waffenroute

Mit diesem Script kannst du deinen Spieler ermöglichen Waffen oder Munition herzustellen 

## Erforderlich

Diese Ressource benötigt die folgenden Ressourcen, um ordnungsgemäß zu funktionieren:

 - EssentialMode
 - esx_extended
 - clip
 - NativeUI
